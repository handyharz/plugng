1. Align on brainstorm and blueprint.
2. Approve screen planning and wireframes.
3. Lock the design system.
4. Build the shared component set.
5. Implement flows in roadmap order.
6. Use the high-fidelity spec, copy guide, and handoff checklist during delivery.
